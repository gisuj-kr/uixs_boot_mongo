Motion

Definition
Motion은 사용자의 인터렉션으로 발생하는 시각적인 변화를 지칭한다.
Principle
수협은행의 친근한 이미지를 전달하기 위해 수협은행에 있는 ‘라온’ 캐릭터를 활용하여 디자인하며,
스플래시이나 로딩같은 대기화면 및 광고의 배너 등에 브랜드 이미지를 전달하기 적합한 화면에 제한적으로
사용할 것을 권장한다.
  1. 1사용자의 행동에 대해 Motion을 제공하여 사용자와 서비스 간 커뮤니케이션을 원활하게 할 수 있다.
  2. 2트랜지션과 애니메이션을 적절히 활용하여 화면의 계층 구조를 파악할 수 있도록 한다.
  3. 3서비스 전반에 걸쳐 일관된 Motion을 사용하여 사용자가 결과를 예측할 수 있도록 한다.
  4. 4적절한 Motion은 사용자가 작업 수행 방식을 이해하도록 도울 수 있다.
  5. 5Motion을 적절히 사용하여 사용자 행동에 시스템이 즉각적으로 반응한다는 인상을 주어야 한다.
  6. 6사용자 행동에 Motion을 제공하여, 필요한 정보로 상요자의 시선을 유도한다.
  7. 7불필요한 Motion 사용으로 사용자의 작업에 방해가 되지 않아야 하며 필요한 경우 드물게 풍부한 애니메이션으로 요소를 강조할 수 있다.
Transition
트랜지션이란 페이지를 이동할 경우 또는 다른 기능을 실행할 경우 화면이 전환되는 효과를 의미한다.
Usage
Animation
애니메이션은 사용자가 액션을 취한 경우 또는 기능 실행이 있어났을 경우 화면에 포함된 요소가 나타내는 효과를 의미한다.
